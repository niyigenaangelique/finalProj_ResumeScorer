# Resume Scoring System for HR Recruitment

A machine learning system that scores resumes for recruitment processes.

## Features
- Automated resume scoring based on multiple criteria
- RESTful API for integration with HR systems
- Simple web interface for HR users
- Feature extraction from resume text
- Model trained on Kaggle resume dataset

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Download and prepare the dataset
3. Train the model: `python train_model.py`
4. Run the API: `python app.py`

## Usage
- Upload resumes via the web interface or API
- Get scores (0-100) with detailed breakdown
- Filter and rank candidates automatically
